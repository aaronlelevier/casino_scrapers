Decorators¶

util.decorators.required(wrapping_functions, patterns_rslt)[source]¶

Used to require 1..n decorators in any view returned by a url tree

..code-block:

# urls.py
urlpatterns += required(
    login_required,
    patterns('',
        (r'^api/', 
            include(api.urls)),
    )
)
Usage:
urlpatterns = required(func,patterns(...)) urlpatterns = required((func,func,func),patterns(...))
Note:
Use functools.partial to pass keyword params to the required decorators. If you need to pass args you will have to write a wrapper function.
Example:

from functools import partial

urlpatterns = required(
partial(login_required,login_url=’/accounts/login/’), patterns(...)

)
